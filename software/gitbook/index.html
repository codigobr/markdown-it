GitBook

GitBook è uno strumento online che ospita siti di documentazione con supporto Markdown.

  1. Markdown
  3. Editori di Markdown
  5. GitBook

Panoramica

Poiché è uno strumento online, GitBook ha un editor Markdown che viene eseguito direttamente nel browser, quindi lo strumento offre molta praticità per creare la documentazione.

Se hai bisogno di un maggiore controllo sui tuoi contenuti, GitBook supporta anche l'integrazione con i repository GitHub.

Con GitBook puoi creare il tuo sito web e poi pubblicarlo utilizzando un dominio personalizzato.

Ma vale la pena ricordare che GitBook è iniziato come un progetto open source e aveva solo una piccola offerta commerciale per raccogliere fondi per l'azienda.

Oggi lo strumento è chiuso e proprietario, quindi tienilo presente se intendi aderire all'uso di GitBook.

Il progetto open source esiste ancora e può essere utilizzato da chiunque, ma GitBook non supporta più questo progetto, quindi verranno elencati solo gli elementi Markdown compatibili con lo strumento proprietario.

Anche se è un'applicazione chiusa, GitBook ha ancora dei vantaggi rispetto ad altri strumenti di questo tipo.

Ad esempio, la costruzione e l'hosting del sito web è responsabilità del programma stesso.

GitBook fornisce un editor Markdown molto intuitivo, con controlli WYSIWYG facili da usare.

Quindi anche gli utenti inesperti si sentono a proprio agio nel modificare i documenti Markdown.

Un'altra caratteristica interessante dello strumento è nascondere i segni di Markdown durante la digitazione.

L'editor può sembrare strano all'inizio, ma col tempo ci si abitua.

GitBook è anche compatibile con copia e incolla, prendi il tuo contenuto Markdown e incollalo direttamente nell'interfaccia dello strumento.

Il sito web ufficiale di GitBook si trova all'indirizzo https://www.gitbook.com/.
