Byword

Byword è un programma compatibile con Markdown per i sistemi macOS e iOS. Si modifica un documento md e lo si esporta in un formato a propria scelta.

  1. Markdown
  3. Editori di Markdown

introduzione

Uno dei vantaggi di Byword è la capacità dell'editor di esportare il suo documento direttamente sulle piattaforme di blog.

Offre anche diverse opzioni per esportare documenti Markdown, inclusi i formati più popolari come HTML, PDF, DOC e LaTeX.

Se hai il servizio iCloud, sarai felice di sapere che usando Byword è possibile sincronizzare i tuoi file Markdown con tutti i dispositivi Apple.

Il Byword può dovere un po 'in termini di bellezza, l'aspetto del programma è molto semplice, ma efficiente nell'editing.

Il sito web ufficiale dello strumento si trova all'indirizzo https://bywordapp.com/
